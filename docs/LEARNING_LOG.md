# Learning Log

This file tracks concepts learned while building RecruiterRadar.

## Phase 0 Concepts

### Git Repository

A Git repository tracks project changes over time. It lets us commit checkpoints, review history, and push the project to GitHub.

### README

The README explains what the project does, why it exists, and how someone can understand or run it.

### `.gitignore`

The `.gitignore` file tells Git which files should not be tracked, such as API keys, dependencies, caches, logs, and generated outputs.

### Environment Variables

Environment variables store configuration and secrets outside the source code. We commit `.env.example` as a template, but never commit the real `.env`.

### CSV

CSV means comma-separated values. It is a simple table format that is easy to read, edit, and process with scripts.

### TypeScript And Node.js

TypeScript is JavaScript with types. Node.js lets JavaScript/TypeScript run outside the browser, which is useful for scripts, API calls, file reading, and backend work.
